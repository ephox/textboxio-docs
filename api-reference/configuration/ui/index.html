The configuration ui property defines options related to the editor UI, and allows for the configuration of the editor toolbar.

Properties

aria-label
 String Text to use for the editor's ARIA label instead of the default 
languages
 Array Specifies available language codes that can be applied to content for internationalization (HTML lang attribute) 
locale
 String Specifies editor UI language 
fonts
 Array Specifies the list of fonts available to the editor 
shortcuts
 Boolean Whether to enable content keyboard shortcuts
toolbar
 Object An object representing the desired toolbar functionality for an editor.

Details

toolbar

For toolbar defaults and other configuration options, see the toolbar documentation page.

aria-label

The default ARIA label is based on the ID attribute of the element that textbox.io replaced. If the element does not have an ID attribute, a 6 digit string is generated based on the current time.

The default label is "Textbox.io Rich Text Editor - <id>" where <id> is the ID attribute value (or generated string). This string is translated into all supported languages.

If a string is provided for this setting, it will be used verbatim with no translation or other change applied to it.

shortcuts

This property is true by default. When set to false, all keyboard shortcuts that alter content formatting will be disabled. The following shortcuts are not affected by this:

  • undo
  • redo
  • help
  • fullscreen
  • focus toolbar
  • open context menu
Icon

Use of this property does not alter the toolbar, it just deactivates the keyboard shortcuts.
