--- layout: default title: editor.macros.removeMacro() ---

Remove a previously added macro by passing its key to editor.macros.removeMacro().

Example

// Remove a previously added macro by key editor.macros.removeMacro('macro_1234567');

Parameters

key String

Key string returned when a macro is created via editor.macros.addSimpleMacro().

No error is thrown if the key does not match a previously added macro.

No return value.

Returns
