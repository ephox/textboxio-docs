Getting started with Textbox.io is super easy via the cloud deployment offering. In this guide you will invoke Textbox.io on a <textarea> as part of a HTML <form>.  When the <form> is submitted, the contents of the Textbox.io editor will be submitted as part of the <form> POST.  

To complete this guide, you will need access to a web server. This guide assumes a web server is running on port 80 on localhost.

Step 1: Signup for an Account with the Textbox.io Cloud Offering

You can signup for a free trial of the Textbox.io Cloud at https://accounts.textbox.io.

Step 2: Add the Domains for Where Textbox.io will be Displayed

For wherever you'll want Textbox.io to appear, you'll need to add the domains for these locations to https://accounts.textbox.io/#domains. The domain http://localhost is enabled by default, allowing you to easier integrate your Textbox.io Cloud instance into your development environments.

Step 3: Add the Textbox.io Cloud <script> Tag to a Web Page

Upon signup to the Textbox.io Cloud, you'll be provided with a <script> tag. You'll need to copy this <script> tag and paste it into the webpage where you will be loading Textbox.io.

Step 4: Invoke Textbox.io as Part of a Web Form

With the script included, you may then invoke Textbox.io on any element (or elements) in your webpage.

Textbox.io lets you identify elements to replace via a CSS3 selector. To add Textbox.io to a page you pass a selector to textboxio.replace().

In this example, you will replace <textarea id="mytextarea"></textarea> with a Textbox.io editor by passing the selector '#mytextarea' to textboxio.replace().

<!DOCTYPE html> <html> <head> <script src="http://clouddelivery.textbox.io/1/getEditor?apiKey=CLIENT_API_KEY"></script> </head> <body> <h1>Textbox.io Getting Started Guide</h1> <form> <textarea id="mytextarea"></textarea> <button type="submit">Submit</button> </form> <script type="text/javascript"> var editor = textboxio.replace('#mytextare`a'); </script> </body> </html>

Try it:

You’ve added Textbox.io to the page - that's all there is to it!

Next, we'll look at retrieving content as part of a <form> POST.

Step 5: Saving Content with a <form> POST

When the form is submitted, the Textbox.io editor mimics the behavior of a normal HTML <textarea> during a form POST. No additional configuration is required.

 

Automatically Configured Spelling and Image Upload Support

Textbox.io Cloud deployments automatically have spelling support and image upload support configured. As a developer, you don't need to configure spelling or image upload when running Textbox.io from the cloud.
